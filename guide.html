const username = document.getElementById("username").value; const password = document.getElementById("password").value; if (username && password.length >= 6) { alert("Login successful!"); } else { alert("Please enter valid details."); } }); ``` **Assignment:** * Add password strength indicator. --- ### 🔹 Session 6: Mini Challenge + Git Setup **Topics:** * Recap quiz * Install Git + GitHub Desktop * Connect VS Code to GitHub **Assignment:** * Push your personal landing page to GitHub * Share link to repo in the learning board --- 📂 **Tab 3 to Tab 7 will be continued in next update.** 📂 Tab 3: Week 3–4 – Projects & Git 🔹 Session 7–9: Mini Project 1 - Responsive Landing Page Topics: Semantic HTML5 Responsive layouts using Flexbox and Media Queries Assignment: Design and build a simple business landing page.
